# Student Defender

Un juego creado como proyecto final para la clase "Programacion orientada a redes" usando libgdx.

Se desarrolla en la Escuela Tecnica N° 35, y los protagonistas son profesores del colegio.

## Prerequisitos

Tener instalado:

* Java
* Gradle

## Como ejecutarlo

* Con VSCode

    1. Clonar el repositorio
    2. Abrir el repositorio en VSCode
    3. Apretar `Ctrl+Shift+B` (`Shift + Command + B` en macOS)
    4. Seleccionar si quiere uno o dos jugadores

* Con los ejecutables

    1. Descargarse los jars [studentDefender-Cliente.jar](/jars/StudentDefender-Cliente.jar) y [studentDefender-Servidor.jar](/jars/StudentDefender-Servidor.jar)
    2. Ejecutar el servidor
    3. Ejecutar uno o dos clientes, dependiendo de con cuantos jugadores quiere jugar
    > Para ejecutar los jars haga doble click en ellos o en la terminal estando en el mismo directorio ejecutar por ejemplo `java -jar  studentDefender-Cliente.jar`
